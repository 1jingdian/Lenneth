# Lenneth Project

### 简介

这是后端项目，利用 api 进行通讯。

### 依赖

- mongodb 2.4+
- elasticsearch 1.3+

### 使用方法

```bash
$ npm install -g supervisor
$ cd Lenneth
$ npm install

$ nohup elasticsearch &
$ supervisor node app
```
